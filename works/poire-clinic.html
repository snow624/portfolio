OVERVIEW

訪問鍼灸・マッサージを行う「ポワール治療院」のWebサイトを制作しました。 通院が難しい高齢者やご家族が安心して利用できるよう、 サービス内容や訪問治療の特徴を分かりやすく伝える構成を意識しています。 地域の利用者が必要な情報にすぐアクセスできるよう、シンプルで見やすいデザインを心がけました。

ROLE

  • サイトデザイン
  • HTML / CSS コーディング
  • レスポンシブ対応
  • サイト公開・運用サポート

TECH STACK

HTML CSS JavaScript

POINTS

① ターゲットに配慮したデザイン

高齢者やご家族が閲覧することを想定し、文字サイズや配色を見やすく調整しました。 直感的に情報が理解できるよう、シンプルなレイアウトを採用しています。

② 情報の整理

訪問治療の内容や利用方法が分かりやすく伝わるよう、情報の優先度を整理し、 ユーザーが必要な情報へ迷わずアクセスできる構成にしました。

③ レスポンシブ対応

スマートフォンからの閲覧が多いことを想定し、 モバイル端末でも読みやすいレイアウトになるよう調整しました。
