OVERVIEW

就職活動で自分のスキルや制作実績を伝えるために制作したポートフォリオサイトです。
デザインからコーディング、サーバー構築までをすべて個人で行い、 GitHub Actionsを使用した自動デプロイ環境を構築しています。
制作物を見やすく伝えることを目的に、シンプルで視認性の高いUI設計を意識しました。

ROLE

  • サイト構成設計 / UI設計
  • デザイン制作
  • HTML / CSS / JavaScript 実装
  • GitHub管理 / CI/CD環境構築
  • GCPサーバー構築・デプロイ

TECH STACK

HTML CSS JavaScript Git GitHub GitHub Actions GCP Compute Engine Apache Let's Encrypt

POINTS

① 見やすさを重視したUI設計

制作実績を短時間で理解できるように、作品一覧・詳細ページの構成を整理しました。 視線の流れを意識し、情報の優先順位を考えたレイアウト設計を行っています。

② 自動デプロイ環境の構築

GitHub Actionsを使用し、mainブランチへpushすると 自動でGCPサーバーへデプロイされるCI/CD環境を構築しました。 手動アップロードを不要にし、開発効率を向上させています。

③ SSH認証エラーの解決

デプロイ時にSSH接続エラーが発生したため、 公開鍵・秘密鍵の設定とGitHub Secretsの管理方法を見直しました。 ログを確認しながら原因を特定し、安定したデプロイ環境を構築しました。

ARCHITECTURE

GitHub → GitHub Actions → GCP → ComputeEngine → Apache → yukikawakami.dev
